DocAppointment

### 📸 Project Preview
![DocAppoint Preview](docappoint-cover.png)

This is a clean and modern hospital services platform designed to help patients easily navigate and discover specialized medical programs. The website provides a clear overview of various departments, from routine check-ups to specialized pediatric care.

Live Site: https://doctor-appointment-manager-lovat.vercel.app

Key Features:
Intuitive User Interface: A clean and professional design that makes navigating hospital services effortless for patients.

Comprehensive Department Listings: Detailed information on various specialties, including Cardiology, Pulmonology, and Neurology.

Dynamic Animations: Uses Lottie animations to create an engaging and friendly user experience.

Specialist Service Overview: Clear descriptions of the medical programs and care options available at the facility.

Fully Responsive Design: Optimized for a seamless experience on all devices, including mobile phones, tablets, and desktops.

Technologies Used:
Next.js

Tailwind CSS

Lucide React

Lottie React
